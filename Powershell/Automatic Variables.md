| Variable | Contains | Example |
| -------- | -------- | ------- |
| `$?` |  | `` |
| `$^` |  | `Get-ChildItem -Attributes Hidden+Directory; Write-Host $^` |
| `$$` |  | `Get-ChildItem -Attributes Hidden+Directory; Write-Host $$` |
| `` |  |
| `` |  |
| `` |  |

- True: `$true`
- False `$false`
- Null: `$null`
- Home: `$Home`
